🤖 AI FAQ Chatbot

An AI-powered FAQ Chatbot built using Python, Streamlit, and Natural Language Processing (NLP). The chatbot understands user queries and provides the most relevant answers from a predefined FAQ knowledge base using TF-IDF Vectorization and Cosine Similarity.

📌 Project Overview

The AI FAQ Chatbot is designed to simulate a customer support assistant by answering frequently asked questions. It processes user input, compares it with stored FAQs, and returns the most relevant response.

✨ Features

- NLP-based text preprocessing
- Intelligent question matching
- TF-IDF Vectorization
- Cosine Similarity scoring
- Interactive chat interface
- Professional dark-themed UI
- Frequently Asked Questions sidebar
- Clear Chat functionality

🛠 Technologies Used

- Python
- Streamlit
- Pandas
- NLTK
- Scikit-learn

🧠 AI Concepts Used

Natural Language Processing (NLP)

Used to preprocess and clean user queries by:

- Tokenization
- Stopword removal
- Text normalization

TF-IDF Vectorization

Converts textual data into numerical vectors that can be compared mathematically.

Cosine Similarity

Measures similarity between user queries and stored FAQ questions to find the best matching answer.

▶️ How to Run

1. Clone the Repository

git clone <repository-url>
cd CodeAlpha_FAQ_Chatbot

2. Install Dependencies

pip install -r requirements.txt

3. Run the Application

py -m streamlit run app.py
